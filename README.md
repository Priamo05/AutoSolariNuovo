# Auto Solari Nuovo - Sito Statico

Questa versione è un sito statico puro, senza backend.

## Funzionalità
- Inserimento misurazioni giornaliere (produzione, consumo, km)
- Dashboard KPI (totale produzione/consumo/km)
- Eliminazione singola riga
- Reset completo dati
- Persistenza nel browser tramite `localStorage`

## Deploy su Cloudflare Pages
1. Collega la repository a Cloudflare Pages
2. Build command: *(vuoto)*
3. Build output directory: `public`
4. Deploy

## Sviluppo locale
Apri direttamente `public/index.html` nel browser oppure servi `public/` con un web server statico.
